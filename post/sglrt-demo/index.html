SGLRT Demo

Last updated on Sep 27, 2020 4 min read R package

SGLRT

SGLRT is a R package implementation of Sequential Generalized Likelihood Ratio (GLR)-like Tests and confidence seqeunces in Paper.

Installation

Currently, the package is under-construction and you can install the development version from GitHub with:

# install.packages("devtools")
devtools::install_github("shinjaehyeok/SGLRT_public")

Example

Let \(X_1, X_2, \dots\) be independent sub Gaussian random variables with para. From the Chernoff-Hoeffiding theorem, we have following inequalities: $$ \mathbb{P}_\mu\left(2n\left(\bar{X}n - \mu\right)^2 \geq \log(2/\alpha)\right) \leq \alpha ~~(\text{sub-Gaussian type}) \
\mathbb{P}\mu\left(n\mathrm{KL}\left(\bar{X}_n, \mu\right) \geq \log(2/\alpha)\right) \leq \alpha ~~(\text{sub-Bernulli type}), $$ for any fixed \(n \in \mathbb{N}\) and \(\alpha \in (0, 1)\) where \(\mathrm{KL}(\mu_1, \mu_0) := \mu_1 \log\left(\frac{\mu_1}{\mu_0}\right) + (1-\mu_1) \log\left(\frac{1-\mu_1}{1-\mu_0}\right)\).

These inequalities can be converted to confidence intervals \(\{\mathrm{CI}_n(\alpha)\}_{n \in \mathbb{N}}\) which capture the underlying mean \(\mu\) with at least \(1-\alpha\) probability: $$ \mathbb{P}_\mu \left( \mu \in \mathrm{CI}_n(\alpha)\right) \geq 1-\alpha,~~\forall \mu \in (0,1), $$ for any fixed \(n \in \mathbb{N}\).

In practice, it could be difficult to pick a proper sample size \(n\) before observing the data or we may want to keep tracking the sequence of confidence intervals for some practical reasons. In this case, however, confidence intervals based on the above Chernoff-Hoeffding inequalities do not have the correct coverage. In fact, from the law of iterated logarithm, we can show $$ \mathbb{P}_\mu \left( \mu \in \mathrm{CI}n(\alpha), \forall n \in \mathbb{N}\right) = 0,~~\forall \mu \in (0,1). $$ If a sequence of random intervals \(\{\mathrm{CS}_n(\alpha)\}_{n \in \mathbb{N}}\) satisfies the following inequality: $$ \mathbb{P}\mu \left( \mu \in \mathrm{CS}_n(\alpha), \forall n \in \mathbb{N}\right) \geq 1-\alpha,~~\forall \mu \in (0,1), $$ it is called a confidence sequence with confidence level \(\alpha\) (e.g, see Howard et al., 2018 and references therein for a comprehensive summary of related literature.)

The main objective of SGLRT package is to implement GLR-like confidence sequences \(\{\mathrm{CS}_n(\alpha)\}_{n \in \mathbb{N}}\) in ADD_CITE which has the following features:

  1. For given level \(\alpha \in (0, 0.5)\) and target interval \([n_{\min}, n_{\max}]\), we can compute a threshold \(g_\alpha >0\) such that the following inequality holds: $$ \mathbb{P}\mu\left(\exists n \in [n{\min}, n_{\max}] : 2n\left(\bar{X}_n - \mu\right)^2 \geq g_\alpha \right) \leq \alpha ~~(\text{sub-Gaussian type}) \
    \mathbb{P}_\mu\left(\exists n \in [n_{\min}, n_{\max}] : n\mathrm{KL}\left(\bar{X}_n, \mu\right) \geq g_\alpha\right) \leq \alpha ~~(\text{sub-Bernulli type}), $$ Of course, we can use the naive threshod \(\log(2 / \alpha) + \log(n_{\max} - n_{\min})\) based on the union bound. However, the threshold \(g_\alpha\) we compute has an exponentailly smaller order: \(g_\alpha = O(\log(2 / \alpha) + \log\log(n_{\max} / n_{\min}))\).

  2. Outside of the target interval \([n_{\min}, n_{\max}]\), the confidence sequence is well-defined and maintain the valid coverage. That is, we have \(\mathbb{P}_\mu \left( \mu \in \mathrm{CS}_n(\alpha), \forall n \in \mathbb{N}\right) \geq 1-\alpha,~~\forall \mu \in (0,1)\).

We say a confidence sequence \(\{\mathrm{CS}_n(\alpha)\}_{n \in \mathbb{N}}\) is time-uniformly close to the Chernoff bound on a finite time interval if it satisfies the first feature above. Based on this property, we can use any existing experimentation system which computes the ordinary pointwise valid Chernoff bound or normal confidence bound to calculate a time-uniform confidence sequnece on the target time interval.

The SGLRT package also implements the discrete mixture confidence sequence Though several tyeps of discrete mixture based confidence sequence have been wel-known in literature, the novel contribution of SGLRT package is to provide a systemic way of design discrete mixture confidence seqeunce which is time-uniformly than the GLR-like one above.

Demo

To demonstrate how to use SGLRT package to compute a time-uniformly valid confidence sequence, let assume we have a stream of independent samples from the Bernoulli distribution with \(\mathbb{P}(X = 1) = 0.5\).

set.seed(100)
# Load package
library(SGLRT)

# Generate 1e+4 samples from a Bernoulli(p) with p = 0.5
n <- 1e+4
mu <- 0.5 

n_vec <- seq(1, n)
x_vec <- rbinom(n, 1, mu) 
x_bar_vec <- cumsum(x_vec) / n_vec

# Set alpha = 0.025 for 95% CI.
alpha <- 0.025

# Set target time interval: [1, 1e+4] and [100, 1e+4]
nmin1 <- 1
nmax1 <- 1e+4
nmin2 <- 2e+2
nmax2 <- 1e+4

Now, let us first compare the poinwise and time-uniform Chernoff bounds in the sub-Gaussian type.

# Compute the pointwise Chernoff bound.
point_G_bound <- sqrt(log(2/alpha) / (2 * n_vec))

# Compute the time-uniform GLR-like bound.
G_out1 <- SGLR_CI_additive(alpha, nmax1, nmin1)
GLR_like_G_bound1 <- sapply(n_vec, G_out1$GLR_like_fn)

G_out2 <- SGLR_CI_additive(alpha, nmax2, nmin2)
GLR_like_G_bound2 <- sapply(n_vec, G_out2$GLR_like_fn)

# Compute the time-uniform discrete mixture bound.
dis_mix_G_bound1 <- sapply(n_vec, G_out1$dis_mix_fn)
dis_mix_G_bound2 <- sapply(n_vec, G_out2$dis_mix_fn)
